步驟 關鍵技術節點 /修改器 具體操作說明
1. 準備模型 Subdivision Surface 按 Ctrl + 2 套用細分表面，讓光影分佈更均勻平滑。
2. 轉換光影 Shader to RGB 將 Eevee 的光影資訊轉換為可編輯的顏色數值。
3. 設定色彩階層 Color Ramp 將插值改為 Constant，手動調整滑桿來產生鮮明的明暗分割。
4. 調整色彩 Hex Code Copy/Paste 在 Color Ramp 中吸取顏色，調整亮度（Value）來製作漸層感。
5. 製作描邊 Solidify Modifier 新增材質、調整 Offset 為 1，並勾選 Flip Normals 來產生輪廓線。
